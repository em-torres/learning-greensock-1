A Tween is a single movement in an animation.
In GSAP, a tween looks like this: 

gsap.to(element, vars object);

  • method refers to the GSAP method you'll like to tweeing with .to(), .from(), .fromTo()
  • element refers to the HTML element you want to animate. You can also use an array of elements to tween multiple DOM elements at the same time. Element is similar to "querySelector" in JS.
  • vars is an object of the properties you want to animate and the values you want to animate them to. You can also include other options like duration, delay, and easing.
